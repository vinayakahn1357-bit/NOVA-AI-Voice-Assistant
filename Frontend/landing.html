✦ What NOVA can do

Built for real conversations

NOVA isn't just a chatbot — it's a thinking, speaking, remembering assistant with production-grade security and enterprise-ready infrastructure.

🎙️
Natural Voice Conversations
Speak naturally and NOVA listens. Powered by real-time speech recognition with smart silence detection. Dedicated Jarvis-style voice interface with animated AI core.
🔀
Dual-Provider Intelligence
Groq for ultra-fast inference, NVIDIA for advanced reasoning. NOVA's smart router automatically picks the best provider — or runs both in hybrid parallel mode.
🧠
Persistent Memory
NOVA learns from every conversation — your interests, preferences, and facts. SQLite WAL-mode backed. She remembers who you are across sessions.
📄
PDF Intelligence NEW
Upload PDFs up to 100MB. TF-IDF semantic retrieval finds relevant chunks. Get structured answers with page-level citations from your documents.
🔐
JWT + Session Auth NEW
Stateless JWT tokens for API clients, session cookies for browsers. Token refresh, Google OAuth bridge, Redis rate limiting — production-grade security.
⚡
Redis Infrastructure NEW
Distributed response caching, sliding-window rate limiting, and session storage. Graceful in-memory fallback when Redis is unavailable.
🎨
5 Voice Personalities
Choose NOVA's voice — the precise NOVA, playful Anya, warm Mitsuri, authoritative Jarvis, or calm Sage. Microsoft Edge Neural TTS quality.
🔄
Streaming + Markdown UPGRADED
Token-by-token SSE streaming with full markdown rendering. Bold, lists, code blocks with syntax highlighting — in both chat and voice views.
🤖
Agent Intelligence
Agent Runner with 5 modes: Normal, Decision, Opinion, Planning, Tool. Workflow Engine for multi-step task orchestration. Built-in calculator, system tools.
